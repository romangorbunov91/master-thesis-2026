\section{Алгоритм с неизменной длительностью измерительного окна, кратной периоду инжектируемого сигнала}\label{sec:Th_const}

\needspace{1\baselineskip}
Принципиальные особенности рассматриваемого алгоритма:
\begin{itemize}
    \item Длительность ${T_h}$ измерительного окна неизменна и равна наибольшему периоду инжектируемого сигнала (${T^{(k)}_h = T^{max}_z}$, ${T^{(k)}_h = const}$).
    \item Период дискретизации ${T_{s}}$ одинаков для всех значений ${T_{z}}$ (${T^{(k)}_s = const}$) и кратен периоду ${T_{d}}$ выполнения операций в системе (${T^{(k)}_s = M \mathord{\cdot} T_d}$); коэффициент децимации ${M = const}$, ${M \in \mathbb{N}}$.
    \item Количество точек ${N \in \mathbb{N}}$ дискретной последовательности, по которым вычисляются коэффициенты разложения в выражении \eqref{eqn:DFT}, неизменно для всех значений ${T^{(k)}_z}$ (${N^{(k)} = const}$),
    \begin{equation*}
        N^{(k)} = \cfrac{T^{(k)}_h} {T^{(k)}_s}.
    \end{equation*}
\end{itemize}

Математическая обработка в соответствии с \eqref{eqn:DFT} накладывает ограничение на период ${T_z}$ инжектируемого сигнала -- требуется, чтобы измерительное окно ${T_h}$ вмещало целое число периодов ${T_z}$ (рисунок~\ref{fig:Th_const}), т.е. для каждого значения ${T^{(k)}_z}$ должно выполняться
\begin{equation}\label{eqn:T_h_const}
    T^{(k)}_h = L^{(k)} \mathord{\cdot} T^{(k)}_z,
\end{equation}
где ${L^{(k)} \in \mathbb{N}}$.

\begin{figure}[H]
    \centering
    \includegraphics [width=1.00\linewidth] {frequencyResponse/Th_const.pdf}

    \caption{Период инжектируемого сигнала при ${T^{(k)}_h = const}$}
    \label{fig:Th_const}
\end{figure}

В соответствии с \eqref{eqn:T_h_const}, частота ${f^{(k)}_z}$ инжектируемого сигнала потенциально принимает значения
\begin{equation*}
    f^{(k)}_{z} = L^{(k)} \cfrac{1}{T_{h}} = L^{(k)} \Delta f_z,
\end{equation*}
где ${\Delta f_z}$ -- шаг приращения частоты инжектируемого сигнала.

Поскольку ${\Delta f_z = const}$, потенциальные значения частот ${f^{(k)}_z}$ равномерно распределены на интервале возможных значений, что \textbf{является главной отличительной особенностью алгоритма} по сравнению с алгоритмами \ref{sec:N_var}, \ref{sec:N_const}.

Составляющей с частотой ${f^{(k)}_z}$ соответствуют коэффициенты ${X^{(k)}\left[l\right]}$, ${Y^{(k)}\left[l\right]}$ при разных значениях ${L^{(k)}}$, определяемых выражением \eqref{eqn:complexFreqResp}.

\needspace{1\baselineskip}
Изложенный алгоритм измерения частотных характеристик \textbf{обладает следующими характеристиками}:
\begin{enumerate}
    \item Минимально возможная частота ${f_z}$ напрямую определяется максимальным значением измерительного окна ${T_h}$, которое ограничено сверху только объемом памяти для хранения массивов-откликов ${x\left[n\right]}$, ${y\left[n\right]}$.
    
    \item Максимально возможная частота ${f_z}$ в пределе составляет ${f_s / 3}$.
    
    \item Требуемый объем памяти для хранения массивов-откликов ${x^{(k)}\left[n\right]}$, ${y^{(k)}\left[n\right]}$ одинаков для всех частот ${f^{(k)}_z}$ инжектируемого сигнала.

    \item Период $T_z$ инжектируемого сигнала потенциально принимает значения от ${T^{min}_z = 3 T_{s}}$ до ${T^{max}_z = T_h}$, но при условии ${T_h / T_s \in \mathbb{N}}$.
    
    \item Шаг приращения частоты ${f_z}$ инжектируемого сигнала (разрешающая способность): ${\Delta f_z = 1 / T_h}$.
\end{enumerate}
