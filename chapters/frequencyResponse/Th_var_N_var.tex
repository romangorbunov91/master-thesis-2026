\section{Алгоритм с изменяемой длительностью измерительного окна, кратной периоду инжектируемого сигнала}\label{sec:fr_optim}

\needspace{1\baselineskip}
{Принципиальные отличия рассматриваемого алгоритма от алгоритма \ref{sec:Th_const}} заключаются в том, что длительность измерительного окна ${T_h}$, длина последовательностей $N$ и период дискретизации ${T_s}$ -- плавающие параметры, соответствующие каждому отдельному значению периода инжектируемого сигнала ${T_z}$. Остаются следующие ограничения:
\begin{itemize}
    \item Возможные значения длительности измерительного окна ${T_h}$ кратны периоду инжектируемого сигнала ${T_z}$ (плавающий коэффициент ${L\in \mathbb{N}_{>0}}$).

    \item Период дискретизации ${T_s}$ кратен периоду выполнения операций в системе ${T_d}$ (плавающий коэффициент децимации ${M\in \mathbb{N}_{>0}}$).
\end{itemize}

Таким образом, в серии из $K$ измерений для каждого значения периода инжектируемого сигнала ${T^{(k)}_z}$, ${k \in \{1, \dots, K\}}$ (рисунок~\ref{fig:Th_var_N_var}), выполняется:
\begin{equation}\label{eqn:windowlength}
    T^{(k)}_h = L^{(k)} \mathord{\cdot} T^{(k)}_z = L^{(k)} \mathord{\cdot} R^{(k)} \mathord{\cdot} T_d = N^{(k)} \mathord{\cdot} \underbrace{M^{(k)} \mathord{\cdot} T_d}_{T^{(k)}_s},
\end{equation}
где ${R^{(k)} \in \mathbb{R}_{>0}}$ -- отношение периода инжектируемого сигнала ${T^{(k)}_z}$ к периоду выполнения операций в системе ${T_d}$.

Аналогично алгоритму \ref{sec:Th_const}, соотношение между периодом инжектируемого сигнала и периодом дискретизации может принимать дробные значения. В совокупности с изменяемой длительностью измерительного окна это обеспечивает на интервале ${f \in \left(0; {f_d / 2} \right)}$ сколь угодно много потенциальных значений частот ${f_z}$ инжектируемого сигнала, что \textbf{является главной отличительной особенностью алгоритма}.

Из равенства \eqref{eqn:windowlength}
\begin{equation}\label{eqn:L_ratio}
    L^{(k)} = \frac{N^{(k)} \mathord{\cdot} M^{(k)}}{R^{(k)}}.
\end{equation}

В связи с тем, что частота инжектируемого сигнала кратна частоте первой гармоники в спектре
\begin{equation*}
    f^{(k)}_z = L^{(k)} \mathord{\cdot} f^{(k)}_h = \cfrac{L^{(k)}}{T^{(k)}_h},
\end{equation*}
множитель ${L^{(k)}}$ ограничен диапазоном
\begin{equation*}
    1 \leq L^{(k)} < N^{(k)} / 2,
\end{equation*}
что, с учетом \eqref{eqn:L_ratio}, накладывает ограничение
\begin{equation*}
    M^{(k)} < R^{(k)} / 2.
\end{equation*}

Таким образом, для каждой $k$-й частоты ${f_z}$ задача сводится к поиску наборов ${\mathbf{s}_{(p)} \in \mathbb{N}_{> 0}^3}$ натуральных чисел ${N, M, L}$, при которых обеспечивается частота ${f_{z(p)}}$ с заданной степенью приближения ${\Delta\epsilon^{*}_{f}}$ к ${f_{z}}$
\begin{equation*}
    \left| f_{z(p)} - f_{z} \right| = \Delta\epsilon_{f(p)} < \Delta\epsilon^{*}_{f},
\end{equation*}
при ограничениях
\begin{equation*}
        N_\text{min} \leq N_{(p)} \leq N_\text{max}, \quad M_{(p)} < R/2.
\end{equation*}

Оптимальным считается такой набор ${\mathbf{s}^{*}}$, в котором параметр $N$ минимален из всех ${\mathbf{s}_{(p)}}$, отвечающих условию
\begin{equation}\label{eqn:freqtol}
    \Delta\epsilon_{f(p)} < \Delta\epsilon^{*}_{f}.
\end{equation}
В случае, когда ни один набор параметров не удовлетворяет условию \eqref{eqn:freqtol}, оптимальным считается набор ${\mathbf{s}^{*}_{(p)}}$, соответствующий минимальному отклонению ${\Delta\epsilon_{f(p)}}$. Таким образом, согласно выражению \eqref{eqn:meas_duration}, минимум количества точек $N$ обеспечивает сокращение продолжительности измерения частотной характеристики.
