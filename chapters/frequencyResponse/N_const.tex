\section{Алгоритм с варьируемой длительностью измерительного окна и неизменным количеством точек}\label{sec:N_const}

Данный алгоритм характеризуется тем, что в серии из $K$ изменений периода ${T_{z}}$ инжектируемого сигнала (${T^{(k)}_{z}}$, ${k \in \left[1;K\right]}$):
\begin{itemize}
    \item Длительность ${T_{h}}$ измерительного окна равна периоду ${T_{z}}$ инжектируемого сигнала (${T^{(k)}_{h} = T^{(k)}_{z}}$, ${T^{(k)}_{h} - var}$).
    \item Количество точек ${N}$ дискретной последовательности, по которым вычисляются коэффициенты разложения в выражении \eqref{eqn:DFT}, неизменно для всех значений периода ${T_{z}}$ (${N^{(k)} = const}$).
    \item Период дискретизации ${T_{s}}$ изменяется в целях обеспечить ${N^{(k)} = const}$ при изменении периода ${T_{z}}$ инжектируемого сигнала (${T^{(k)}_{s} - var}$).
\end{itemize}

Постоянство $N$ при изменении периода ${T_{z}}$ инжектируемого сигнала обеспечивается децимацией (прореживанием) дискретных последовательностей ${x'[n]}$, ${y'[n]}$.

Выражениями ${x'[n]}$, ${y'[n]}$ обозначены дискретные последовательности, формируемые в результате дискретизации откликов системы ${x_{c}(t)}$, ${y_{c}(t)}$ непосредственно с периодом ${T_{d}}$ выполнения операций в системе.

Дискретные последовательности ${x^{(k)}[n]}$, ${y^{(k)}[n]}$ формируются в результате децимации ${x'^{(k)}[n]}$, ${y'^{(k)}[n]}$ с коэффициентом децимации ${M^{(k)} \in \mathbb{N}}$
\begin{equation*}
    \begin{split}
        &x^{(k)}[n] = x'^{(k)}[n \mathord{\cdot} M^{(k)}] = x^{(k)}_{c} (n \mathord{\cdot} \underbrace{M^{(k)} \mathord{\cdot} T_d}_{T^{(k)}_{s}}) \\
        &y^{(k)}[n] = y'^{(k)}[n \mathord{\cdot} M^{(k)}] = y^{(k)}_{c} (n \mathord{\cdot} \underbrace{M^{(k)} \mathord{\cdot} T_d}_{T^{(k)}_{s}}),
    \end{split}
\end{equation*}
так, что период инжектируемого сигнала всегда кратен периоду ${T_{d}}$ выполнения операций в системе:
\begin{equation*}
    T^{(k)}_{z} = N^{(k)} \mathord{\cdot} \underbrace{M^{(k)} \mathord{\cdot} T_d}_{T^{(k)}_{s}}.
\end{equation*}

Дискретные последовательности ${x^{(k)}[n]}$, ${y^{(k)}[n]}$ формируются по правилу \eqref{eqn:sampleRule} фактически с эквивалентным периодом дискретизации ${T^{(k)}_{s} = M^{(k)} \mathord{\cdot} T_d}$ (рисунок~\ref{fig:N_const}). Под эквивалентным понимается такой период дискретизации, при котором формировалась бы точно такая же дискретная последовательность.

Измерительное окно ${T^{(k)}_{h} = T^{(k)}_{z}}$ при всех $k$, поэтому, также как и в алгоритме \ref{sec:N_var}, составляющей с частотой ${f^{(k)}_{z}}$ соответствуют коэффициенты ${X^{(k)}\left[l\right]}$, ${Y^{(k)}\left[l\right]}$ всегда при ${l = 1}$, т.е. коэффициент передачи динамического звена на частоте ${f^{(k)}_{z}}$ определяется выражением \eqref{eqn:complexFreqResp} при ${L^{(k)} = 1}$.

\textbf{Принципиальное отличие} от алгоритма \ref{sec:N_var} связано с потенциально меньшим размером $N$ массивов-откликов ${x\left[n\right]}$, ${y\left[n\right]}$ (${N \in \mathbb{N}}$ -- независимая переменная, ${N > 2}$).

Таким образом, изложенному алгоритму измерения частотных характеристик с последовательной сменой частоты ${f_{z}}$ инжектируемого сигнала в серии из $K$ измерений \textbf{характерны следующие особенности}:
\begin{enumerate}
    \item Минимально возможная частота ${f_{z}}$ напрямую определяется максимальным значением измерительного окна ${T_{h}}$, причем требуемый объем памяти для хранения массивов-откликов ${x^{(k)}\left[n\right]}$, ${y^{(k)}\left[n\right]}$ одинаков для всех длительностей ${T^{(k)}_{h}}$ и частот ${f^{(k)}_{z}}$.
    
    \item Максимально возможная частота ${f_{z}}$ составляет ${f_{d} / N}$.
      
    \item Шаг приращения периода ${T_{z}}$ инжектируемого сигнала (разрешающая способность) в $N$-раз больше, в сравнении с алгоритмом \ref{sec:N_var} (${\Delta T_{z} = N \mathord{\cdot} T_{d}}$).
    
    \item Грубый шаг приращения периода обуславливает еще более неравномерное распределение частот. Например, в диапазоне частот ${f \in \left[{f_{d} / 100}; {f_{d} / 10} \right]}$ при ${N = 10}$ частота ${f_{z}}$ инжектируемого сигнала может принимать значения всего из 10-ти возможных: \\
    ${f_{z} = \left({f_{d} / 100}, {f_{d} / 90}, ..., {f_{d} / 10} \right)}$.
\end{enumerate}
