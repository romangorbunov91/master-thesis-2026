\section{Алгоритм с изменяемой длительностью измерительного окна, равной периоду инжектируемого сигнала, и децимацией}\label{sec:N_const}

\needspace{1\baselineskip}
Принципиальные особенности рассматриваемого алгоритма:
\begin{itemize}
    \item Длительность измерительного окна ${T_h}$ равна периоду инжектируемого сигнала ${T_z}$.
    \item Период дискретизации ${T_s}$ кратен периоду выполнения операций в системе ${T_d}$, но изменяется в целях обеспечить неизменную длину $N$ дискретных последовательностей для всех значений периода инжектируемого сигнала ${T_z}$.
\end{itemize}

Постоянство $N$ при изменении периода инжектируемого сигнала ${T_z}$ обеспечивается децимацией (прореживанием) дискретных последовательностей ${x'[n]}$, ${y'[n]}$ с коэффициентом децимации ${M^{(k)} \in \mathbb{N}}$:
\begin{equation*}
    \begin{split}
        &x^{(k)}[n] = x'^{(k)}[n \mathord{\cdot} M^{(k)}] = x^{(k)}_{c} (n \mathord{\cdot} \underbrace{M^{(k)} \mathord{\cdot} T_d}_{T^{(k)}_{s}}) \\
        &y^{(k)}[n] = y'^{(k)}[n \mathord{\cdot} M^{(k)}] = y^{(k)}_{c} (n \mathord{\cdot} \underbrace{M^{(k)} \mathord{\cdot} T_d}_{T^{(k)}_{s}}),
    \end{split}
\end{equation*}
Выражениями ${x'[n]}$, ${y'[n]}$ обозначены дискретные последовательности, формируемые в результате дискретизации откликов системы ${x_{c}(t)}$, ${y_{c}(t)}$ непосредственно с периодом выполнения операций в системе ${T_d}$.

Таким образом, в серии из $K$ изменений для каждого значения периода инжектируемого сигнала ${T^{(k)}_z}$, ${k \in \{1, \dots, K\}}$ (рисунок~\ref{fig:N_const}), выполняется:
так, что период инжектируемого сигнала всегда :
\begin{equation*}
    T^{(k)}_{z} = T^{(k)}_h = N^{(k)} \mathord{\cdot} \underbrace{M^{(k)} \mathord{\cdot} T_d}_{T^{(k)}_{s}}.
\end{equation*}

Также как и в алгоритме \ref{sec:N_var}, составляющей с частотой ${f^{(k)}_{z}}$ соответствуют коэффициенты ${X^{(k)}\left[l\right]}$, ${Y^{(k)}\left[l\right]}$ всегда при ${l = 1}$, т.е. коэффициент передачи динамического звена на частоте ${f^{(k)}_{z}}$ определяется выражением \eqref{eqn:complexFreqResp} при ${L^{(k)} = 1}$.

\textbf{Принципиальное отличие} от алгоритма \ref{sec:N_var} связано с потенциально меньшей длиной $N$ массивов-откликов ${x\left[n\right]}$, ${y\left[n\right]}$.

\needspace{1\baselineskip}
Таким образом, изложенный алгоритм измерения частотных характеристик \textbf{обладает следующими характеристиками}:
\begin{enumerate}
    \item Минимально возможная частота ${f_{z}}$ напрямую определяется максимальным значением измерительного окна ${T_{h}}$, причем требуемый объем памяти для хранения массивов-откликов ${x^{(k)}\left[n\right]}$, ${y^{(k)}\left[n\right]}$ одинаков для всех длительностей ${T^{(k)}_{h}}$ и частот ${f^{(k)}_{z}}$.
    
    \item Максимально возможная частота ${f_{z}}$ составляет ${f_{d} / N}$.
      
    \item Шаг приращения периода ${T_{z}}$ инжектируемого сигнала (разрешающая способность) в $N$-раз больше, в сравнении с алгоритмом \ref{sec:N_var} (${\Delta T_{z} = N \mathord{\cdot} T_{d}}$).
    
    \item Грубый шаг приращения периода обуславливает еще более неравномерное распределение частот. Например, в диапазоне частот ${f \in \left[{f_{d} / 100}; {f_{d} / 10} \right]}$ при ${N = 10}$ частота ${f_{z}}$ инжектируемого сигнала может принимать значения всего из 10-ти возможных: \\
    ${f_{z} = \left({f_{d} / 100}, {f_{d} / 90}, ..., {f_{d} / 10} \right)}$.
\end{enumerate}
