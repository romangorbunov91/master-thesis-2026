\section{Алгоритм ${T_{h} - var}$, ${N - var}$}\label{sec:N_var}

Данный алгоритм характеризуется тем, что в серии из $K$ изменений периода ${T_{z}}$ инжектируемого сигнала (${T^{(k)}_{z}}$, ${k \in \left[1;K\right]}$):
\begin{itemize}
    \item Длительность ${T_{h}}$ измерительного окна равна периоду ${T_{z}}$ инжектируемого сигнала (${T^{(k)}_{h} = T^{(k)}_{z}}$, ${T^{(k)}_{h} - var}$).
    \item Период дискретизации ${T_{s}}$ одинаков для всех значений ${T_{z}}$ и равен периоду ${T_{d}}$ выполнения операций в системе (${T^{(k)}_{s} = T_{d}}$, ${T^{(k)}_{s} = const}$).
    \item Количество точек $N$ дискретной последовательности, по которым вычисляются коэффициенты разложения в выражении \eqref{eqn:DFT}, изменяется пропорционально ${T_{z}}$ (${N^{(k)} - var}$).
\end{itemize}

Период инжектируемого сигнала
\begin{equation*}
    T^{(k)}_{z} = N^{(k)} \cdot T^{(k)}_{s} = N^{(k)} \cdot T_{d}
\end{equation*}
кратен периоду ${T_{d}}$ выполнения операций в системе (рис.~\ref{fig:N_var}).

\begin{figure}[H]
	\centering
	\includegraphics [width=0.85\linewidth] {frequencyResponse/N_var.drawio.png}

	\caption{Размер ${N^{(k)}}$ дискретной последовательности при изменении периода ${T^{(k)}_{z}}$ инжектируемого сигнала}
	\label{fig:N_var}
\end{figure}

В связи с тем, что ${T^{(k)}_{h} = T^{(k)}_{z}}$ составляющей с частотой ${f^{(k)}_{z}}$ соответствуют коэффициенты ${X^{(k)}\left[l\right]}$, ${Y^{(k)}\left[l\right]}$ всегда при ${l = 1}$, т.е. коэффициент передачи динамического звена на частоте ${f^{(k)}_{z}}$ определяется выражением \eqref{eqn:complexFreqResp} при ${L^{(k)} = 1}$.

Таким образом, изложенному алгоритму измерения частотных характеристик с последовательной сменой частоты ${f_{z}}$ инжектируемого сигнала в серии из $K$ измерений \textbf{характерны следующие особенности}:
\begin{enumerate}
    \item Минимально возможная частота ${f_{z}}$ напрямую определяется максимальным значением измерительного окна ${T_{h}}$, которое ограничено сверху только объемом памяти для хранения массивов-откликов ${x\left[n\right]}$, ${y\left[n\right]}$.
    
    \item Максимально возможная частота ${f_{z}}$ в пределе составляет ${f_{d} / 3}$.
    
    \item В памяти достаточно хранить ${N^{(k)} = f_{d} / f^{(k)}_{z}}$-значений каждого массива-отклика ${x^{(k)}\left[n\right]}$, ${y^{(k)}\left[n\right]}$, причем величина ${N}$ не является постоянной и уменьшается с ростом частоты ${f_{z}}$ вплоть до ${N = 3}$ при ${f_{z} \rightarrow f_{d} / 3}$.
    
    \item Шаг приращения периода ${T_{z}}$ инжектируемого сигнала (разрешающая способность): ${\Delta T_{z} = T_{d}}$.
    
    \item Потенциальные значения частот ${f_{z}}$ инжектируемого сигнала неравномерно распределены на интервале ${f \in \left(0; {f_{d} / 2} \right)}$ возможных значений:
    \begin{itemize}
        \item На полуинтервале ${f \in \left[{f_{d} / 10}; {f_{d} / 2} \right)}$ частота ${f_{z}}$ может принимать значения всего из 8-ми возможных: \\
        ${f_{z} = \left({f_{d} / 10}, {f_{d} / 9}, ..., {f_{d} / 3} \right)}$.
        \item На полуинтервале ${f \in \left[{f_{d} / 100}; {f_{d} / 10} \right)}$ -- одно из 90 возможных: \\
        ${f_{z} = \left({f_{d} / 100}, {f_{d} / 99}, ..., {f_{d} / 11} \right)}$.
        \item На интервале ${f \in \left(0; {f_{d} / 100} \right)}$ -- сколь угодно много значений, ограниченное исключительно объемом памяти для хранения массивов-откликов ${x\left[n\right]}$, ${y\left[n\right]}$.
    \end{itemize}
\end{enumerate}
