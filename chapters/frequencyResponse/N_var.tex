\section{Алгоритм с изменяемой длительностью измерительного окна, равной периоду инжектируемого сигнала}\label{sec:N_var}

\needspace{1\baselineskip}
Принципиальные особенности рассматриваемого алгоритма:
\begin{itemize}
    \item Длительность измерительного окна ${T_h}$ равна периоду инжектируемого сигнала ${T_z}$.
    \item Период дискретизации ${T_s}$ одинаков для всех значений периода инжектируемого сигнала ${T_z}$ и равен периоду выполнения операций в системе ${T_d}$.
    \item Длина $N$ дискретных последовательностей, по которым вычисляются коэффициенты разложения в выражении \eqref{eqn:DFT}, изменяется пропорционально периоду инжектируемого сигнала ${T_z}$.
\end{itemize}

Таким образом, в серии из $K$ изменений для каждого значения периода инжектируемого сигнала ${T^{(k)}_z}$, ${k \in \{1, \dots, K\}}$ (рисунок~\ref{fig:N_var}), выполняется:
\begin{equation*}
    T^{(k)}_z = T^{(k)}_h = N^{(k)} \mathord{\cdot} T^{(k)}_s = N^{(k)} \mathord{\cdot} T_d.
\end{equation*}

В связи с тем, что ${T^{(k)}_h = T^{(k)}_z}$, составляющей с частотой ${f^{(k)}_z}$ соответствуют коэффициенты ${X^{(k)}\left[l\right]}$, ${Y^{(k)}\left[l\right]}$ всегда при ${l = 1}$, т.е. коэффициент передачи динамического звена на частоте ${f^{(k)}_z}$ определяется выражением \eqref{eqn:complexFreqResp} при ${L^{(k)} = 1}$.

\needspace{1\baselineskip}
Изложенный алгоритм измерения частотных характеристик \textbf{обладает следующими характеристиками}:
\begin{enumerate}
    \item Минимально возможная частота ${f_z}$ напрямую определяется максимальным значением измерительного окна ${T_h}$, которое ограничено сверху только объемом памяти ${N_\text{max}}$ для хранения массивов-откликов ${x\left[n\right]}$, ${y\left[n\right]}$.
    
    \item В памяти достаточно хранить ${N^{(k)} = f_d / f^{(k)}_z}$-значений каждого массива-отклика ${x^{(k)}\left[n\right]}$, ${y^{(k)}\left[n\right]}$, причем величина $N$ не является постоянной и уменьшается с ростом частоты ${f_z}$ вплоть до ${N = N_\text{min}}$.
    
    \item В случае ${N_\text{min}=3}$ максимально возможная частота ${f_z}$ ограничена сверху значением ${f_d / 3}$.
    
    \item Шаг приращения периода ${T_z}$ инжектируемого сигнала (разрешающая способность): ${\Delta T_z = T_d}$.
    
    \item Потенциальные значения частот ${f_z}$ инжектируемого сигнала неравномерно распределены на интервале ${f \in \left(0; {f_d / 2} \right)}$ возможных значений:
    \begin{itemize}
        \item На полуинтервале ${f \in \left[{f_d / 10}; {f_d / 2} \right)}$ частота ${f_z}$ может принимать значения всего из 8-ми возможных: \\
        ${f_z = \left({f_d / 10}, {f_d / 9}, ..., {f_d / 3} \right)}$.
        \item На полуинтервале ${f \in \left[{f_d / 100}; {f_d / 10} \right)}$ -- одно из 90 возможных: \\
        ${f_z = \left({f_d / 100}, {f_d / 99}, ..., {f_d / 11} \right)}$.
        \item На интервале ${f \in \left(0; {f_d / 100} \right)}$ -- сколь угодно много значений, ограниченное исключительно объемом памяти ${N_\text{max}}$ для хранения массивов-откликов ${x\left[n\right]}$, ${y\left[n\right]}$.
    \end{itemize}
\end{enumerate}
