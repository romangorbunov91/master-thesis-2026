\section{Алгоритм с изменяемой длительностью измерительного окна, равной периоду инжектируемого сигнала}\label{sec:N_var}

\needspace{1\baselineskip}
Принципиальные особенности рассматриваемого алгоритма:
\begin{itemize}
    \item Длительность измерительного окна ${T_h}$ равна периоду инжектируемого сигнала ${T_z}$.
    \item Длина $N$ последовательностей ${x\left[n\right]}$, ${y\left[n\right]}$, по которым вычисляются коэффициенты разложения в выражении \eqref{eqn:DFT}, изменяется пропорционально периоду инжектируемого сигнала ${T_z}$.
    \item Период дискретизации ${T_s}$ одинаков для всех значений периода инжектируемого сигнала ${T_z}$ и равен периоду выполнения операций в системе ${T_d}$.
\end{itemize}

Таким образом, в серии из $K$ изменений для каждого значения периода инжектируемого сигнала ${T^{(k)}_z}$, ${k \in \{1, \dots, K\}}$ (рисунок~\ref{fig:N_var}), выполняется:
\begin{equation*}
    T^{(k)}_h = T^{(k)}_z = N^{(k)} \mathord{\cdot} T^{(k)}_s = N^{(k)} \mathord{\cdot} T_d.
\end{equation*}

Длина массивов-откликов пропорциональна периоду инжектируемого сигнала
\begin{equation}\label{eqn:seq_len}
    N^{(k)} = \cfrac{T^{(k)}_z}{T_s},
\end{equation}
что представляет одно из ключевых ограничений рассматриваемого алгоритма в области измерений низких частот.

В связи с тем, что длительность измерительного окна ${T^{(k)}_h}$ равна периоду инжектируемого сигнала ${T^{(k)}_z}$, составляющей с частотой ${f^{(k)}_z}$ соответствуют коэффициенты ${X^{(k)}\left[l\right]}$, ${Y^{(k)}\left[l\right]}$ всегда при ${l = 1}$, т.е. коэффициент передачи динамического звена на частоте ${f^{(k)}_z}$ определяется выражением \eqref{eqn:complexFreqResp} при ${L^{(k)} = 1}$.

\needspace{1\baselineskip}
Изложенному алгоритму измерения характерно следующее:
\begin{enumerate}
    \item Максимально возможный период инжектируемого сигнала ${T_z}$ соответствует максимальному значению измерительного окна ${T_h}$, которое ограничено только параметром ${N_\text{max}}$.
    
    \item Шаг приращения периода инжектируемого сигнала ${T_z}$ (разрешающая способность): ${\Delta T_z = T_d}$.
    
    \item Длина $N$ каждого массива-отклика ${x\left[n\right]}$, ${y\left[n\right]}$ не является постоянной, но пропорциональна периоду инжектируемого сигнала ${T_z}$ (выражение \eqref{eqn:seq_len}).
    
    \item В случае ${N_\text{min}=3}$ максимально возможная частота ${f_z}$ ограничена сверху значением ${f_d / 3}$.
    
    \item Потенциальные значения частот ${f_z}$ инжектируемого сигнала неравномерно распределены на интервале ${f \in \left(0; {f_d / 2} \right)}$ возможных значений:
    \begin{itemize}
        \item На полуинтервале ${f \in \left[{f_d / 10}; {f_d / 2} \right)}$ частота ${f_z}$ может принимать значения всего из 8-ми возможных: \\
        ${f_z = \left({f_d / 10}, {f_d / 9}, ..., {f_d / 3} \right)}$.
        \item На полуинтервале ${f \in \left[{f_d / 100}; {f_d / 10} \right)}$ -- 90 возможных значений: \\
        ${f_z = \left({f_d / 100}, {f_d / 99}, ..., {f_d / 11} \right)}$.
        \item На интервале ${f \in \left(0; {f_d / 100} \right)}$ -- сколь угодно много значений, ограниченное параметром ${N_\text{max}}$.
    \end{itemize}
\end{enumerate}
