\section{Алгоритмы измерения частотных характеристик}

\needspace{1\baselineskip}
\subsection{Алгоритмы с моногармоническим возмущающим сигналом}

Метод sine-sweep основан на последовательном возбуждении системы синусоидальными сигналами разной частоты и измерении установившегося отклика \cite{ridley2006frequency}. В отличие от широкополосных методов (PRBS, multisine), sine-sweep обеспечивает высокое отношение сигнал/шум на каждой частоте за счёт узкополосного анализа.

На вход системы (канал управления $d[n]$) подаётся синусоидальная последовательность:
\begin{equation}
    p[n] = A \mathord{\cdot} \sin\left(2\pi f_k n T_s + \phi_0\right), \quad n = 0,1,\dots,N_k-1,
    \label{eq:sine_pert}
\end{equation}
где:
\begin{itemize}
    \item $A$ -- амплитуда возмущения (обычно $0.5\%$--$5\%$ от номинального $D$);
    \item $f_k$ -- текущая тестовая частота, $f_k \in [f_{\min}, f_{\max}]$;
    \item $T_s$ -- период дискретизации ($T_s = T_{sw}$ для синхронизации с ШИМ);
    \item $N_k$ -- число периодов усреднения на частоте $f_k$.
\end{itemize}

После переходного процесса ($\sim 3$--$5$ постоянных времени системы) регистрируется выходной сигнал $y[n]$ (напряжение/ток). Для подавления шума и гармоник применяется когерентное усреднение по $M$ периодам:
\begin{equation}
    \bar{y}[m] = \frac{1}{M} \sum_{\ell=0}^{M-1} y\left[m + \ell \mathord{\cdot} N_{\text{per}}\right], 
    \quad m = 0,1,\dots,N_{\text{per}}-1,
    \label{eq:coherent_avg}
\end{equation}
где $N_{\text{per}} = f_s / f_k$ -- число отсчётов на период.

Амплитуда и фаза отклика извлекаются через корреляцию с опорными синусом и косинусом (метод квадратурной демодуляции):
\begin{align}
    Y_{\cos}(f_k) &= \frac{2}{N} \sum_{n=0}^{N-1} \bar{y}[n] \mathord{\cdot} \cos(2\pi f_k n T_s), \label{eq:demod_cos} \\
    Y_{\sin}(f_k) &= \frac{2}{N} \sum_{n=0}^{N-1} \bar{y}[n] \mathord{\cdot} \sin(2\pi f_k n T_s), \label{eq:demod_sin} \\
    G_{vd}(e^{j\omega_k T_s}) &= \frac{Y_{\cos}(f_k) - j Y_{\sin}(f_k)}{A}, \label{eq:transfer_func}
\end{align}
где $\omega_k = 2\pi f_k$. Формула \eqref{eq:transfer_func} даёт оценку передаточной функции на частоте $f_k$.

Для минимизации нелинейных искажений и максимизации ОСШ амплитуда $A$ может подбираться автоматически:
\begin{equation}
    A_{\text{opt}} = \arg\min_{A} \left\{ \sigma_{\text{noise}}(A) \; \big| \; 
    \frac{\|Y(f_k)\|}{\sigma_{\text{noise}}} \geq \text{SNR}_{\min} \right\},
    \label{eq:adaptive_amp}
\end{equation}
где $\sigma_{\text{noise}}$ оценивается по «хвосту» импульсной характеристики или в полосе выше $f_k$.

\begin{enumerate}
    \item Задать диапазон $[f_{\min}, f_{\max}]$ и закон развёртки:
    \begin{itemize}
        \item \textbf{Линейный}: $f_{k+1} = f_k + \Delta f$;
        \item \textbf{Логарифмический}: $f_{k+1} = f_k \mathord{\cdot} r$, $r = (f_{\max}/f_{\min})^{1/(K-1)}$.
    \end{itemize}
    \item Для каждой $f_k$:
    \begin{enumerate}
        \item Сгенерировать $p[n]$ по \eqref{eq:sine_pert};
        \item Дождаться установления ($\sim 3\tau_{\text{sys}}$);
        \item Собрать $y[n]$, применить \eqref{eq:coherent_avg};
        \item Вычислить $G_{vd}(e^{j\omega_k T_s})$ по \eqref{eq:demod_cos}--\eqref{eq:transfer_func};
        \item Проверить критерии сходимости (ОСШ, стабильность амплитуды).
    \end{enumerate}
    \item Построить АЧХ/ФЧХ: $|G_{vd}|_{\text{dB}} = 20\log_{10}|G_{vd}|$, $\angle G_{vd} = \arg(G_{vd})$.
\end{enumerate}

\needspace{1\baselineskip}
\textbf{Практические рекомендации для реализации}
\begin{itemize}
    \item \textbf{Синхронизация}: $f_k$ должна быть кратной $f_s/N_{\text{per}}$ для исключения утечки спектра.
    \item \textbf{Длительность измерения}: $T_{\text{meas}}(f_k) \geq \max\left( \frac{5}{f_k}, \frac{3}{\zeta \omega_n} \right)$, где $\zeta$, $\omega_n$ -- параметры доминирующей моды.
    \item \textbf{Подавление гармоник}: использовать оконные функции (Hann, Blackman) при вычислении \eqref{eq:demod_cos}--\eqref{eq:demod_sin}, если $N_{\text{per}}$ нецелое.
    \item \textbf{Адаптивная развёртка}: увеличивать время усреднения $M$ на частотах с низким ОСШ или вблизи резонансов.
\end{itemize}

\needspace{1\baselineskip}
\subsection{Алгоритмы с полигармоническим возмущающим сигналом}
В качестве возбуждения используется псевдослучайная бинарная последовательность (PRBS) (рис.~\ref{fig:PRBS}), близкая по спектру к белому шуму \cite{shirazi2009autotuning}. Сигнал вводится в управляющий контур преобразователя:
\begin{itemize}
    \item В цикл управления скважностью (duty cycle);
    \item В опорные сигналы тока и напряжения.
\end{itemize}
Такое распределение гарантирует, что тестовый сигнал не будет подавлен регуляторами в широком частотном диапазоне, обеспечивая возбуждение системы на частотах выше и ниже полосы пропускания контуров управления \cite{riccobono2017stability}.

\begin{figure}[H]
	\centering
	\includegraphics [width=1.00\linewidth] {review/PRBS.png}

	\caption{Алгоритм PRBS}
	\label{fig:PRBS}
\end{figure}

В процессе возмущения измеряются мгновенные значения напряжения $v(t)$ и тока $i(t)$. С помощью быстрого преобразования Фурье (БПФ) получают частотные спектры:
\begin{equation}
    V(j\omega) = \mathcal{F}\{v(t)\}, \quad I(j\omega) = \mathcal{F}\{i(t)\}.
\end{equation}
Импеданс подсистемы вычисляется как:
\begin{equation}
    Z(j\omega) = \frac{V(j\omega)}{I(j\omega)}.
\end{equation}
Для получения параметрической модели (передаточной функции) к непараметрической оценке применяется алгоритм наименьших квадратов, позволяющий аппроксимировать импеданс моделью заданного порядка (число полюсов/нулей).

\needspace{1\baselineskip}
Измеренные импедансы и передаточные функции используются для автоматической перенастройки локальных контроллеров преобразователей. Алгоритм адаптации работает в цикле:
\begin{enumerate}
    \item \textbf{Возмущение:} Ввод PRBS-сигнала в контур управления.
    \item \textbf{Сбор данных:} Запись откликов $v(t), i(t)$ в течение времени $T_{\text{IDENT}}$.
    \item \textbf{Анализ:} БПФ-обработка и вычисление частотной характеристики $G(j\omega)$.
    \item \textbf{Аппроксимация:} Подбор параметрической модели с заданным числом полюсов и нулей.
    \item \textbf{Синтез:} Расчёт новых коэффициентов регулятора (например, методом Internal Model Control) с учётом пользовательских ограничений по устойчивости и быстродействию.
    \item \textbf{Обновление:} Загрузка новых коэффициентов в цифровой контроллер (FPGA/DSP).
\end{enumerate}
Полный цикл адаптации занимает время $T_{\text{TOTAL}} \propto T_{\text{IDENT}}$.

В методе минимизации крест-фактора полигармонических (multisine) сигналов на основе аппроксимации нормы Чебышёва \cite{Zhang2025}, \cite{guillaume1991crest} полигармонический возбуждающий сигнал с заданным спектром мощности определяется в дискретной временной области как:
\begin{equation}
    e(n, \boldsymbol{\phi}) = \sum_{m=1}^{M} Z_m \sin(2\pi f_m T n + \phi_m), \quad n = 0, 1, \dots, N-1,
    \label{eq:multisine}
\end{equation}
где $Z_m$, $f_m$ и $\phi_m$ -- амплитуда, частота и фаза $m$-й гармоники соответственно; $M$ -- число гармоник, $T$ -- период дискретизации, $\boldsymbol{\phi} = [\phi_1, \dots, \phi_M]^\top \in \mathbb{R}^M$ -- вектор оптимизируемых фаз.

Крест-фактор (Crest Factor, CF) характеризует отношение пикового значения сигнала к его среднеквадратичному (RMS) уровню:
\begin{equation}
CF(e) = \frac{\max_n |e(n, \boldsymbol{\phi})|}{\text{rms}(e(n, \boldsymbol{\phi}))} = \frac{\|e(n, \boldsymbol{\phi})\|_\infty}{\text{rms}(e(n, \boldsymbol{\phi}))}.
\label{eq:cf_def}
\end{equation}
Поскольку амплитуды $Z_m$ фиксируются на этапе проектирования, значение $\text{rms}(e)$ остаётся постоянным. Следовательно, задача минимизации CF эквивалентна минимизации нормы Чебышёва ($L_\infty$-нормы) сигнала:
\begin{equation}
\min_{\boldsymbol{\phi}} \|e(n, \boldsymbol{\phi})\|_\infty.
\label{eq:optim_problem}
\end{equation}

Прямая градиентная оптимизация $L_\infty$-нормы затруднена из-за её негладкости (производная не существует в точках глобального максимума). В работе предлагается аппроксимировать $L_\infty$-норму пределом обобщённой $L_p$-нормы при $p \to \infty$:
\begin{equation}
    L_p(e(n, \boldsymbol{\phi})) = \left( \frac{1}{N} \sum_{n=1}^{N} |e(n, \boldsymbol{\phi})|^p \right)^{1/p}.
    \label{eq:lp_norm}
\end{equation}
Оптимизация выполняется итеративно по возрастающей последовательности чётных значений $p \in \{4, 8, 16, 32, \dots, p_{\max}\}$. Оптимальный вектор фаз $\boldsymbol{\phi}^{(p)}$, полученный при текущем $p$, используется в качестве начального приближения для минимизации при следующем $p$, что обеспечивает плавную и устойчивую сходимость к решению.

Для каждого фиксированного $p$ задача сводится к задаче нелинейного метода наименьших квадратов. Вводится вектор невязок $\mathbf{E} \in \mathbb{R}^N$:
\begin{equation}
\mathbf{E} = \left[ e(1, \boldsymbol{\phi})^h,\; e(2, \boldsymbol{\phi})^h,\; \dots,\; e(N, \boldsymbol{\phi})^h \right]^\top, \quad h = \frac{p}{2}.
\label{eq:residual}
\end{equation}
Целевая функция принимает вид $\Lambda_p(\boldsymbol{\phi}) = \|\mathbf{E}\|^2$.

На $s$-й итерации алгоритма вычисляется матрица Якоби $\hat{\mathbf{J}}_s \in \mathbb{R}^{N \times M}$. Её элементы определяются аналитически:
\begin{equation}
\left[ \hat{\mathbf{J}}_s \right]_{n,u} = \frac{\partial \hat{e}(n, \hat{\boldsymbol{\phi}}_s)^h}{\partial \hat{\phi}_{u,s}} = h \, \hat{e}(n, \hat{\boldsymbol{\phi}}_s)^{h-1} Z_u \cos(2\pi f_u T n + \hat{\phi}_{u,s}), \quad u=1,\dots,M.
\label{eq:jacobian}
\end{equation}
Шаг обновления вектора фаз вычисляется по классической формуле Левенберга-Марквардта:
\begin{equation}
\mathbf{d}_s = - \left( \hat{\mathbf{J}}_s^\top \hat{\mathbf{J}}_s + \lambda_s \mathbf{I} \right)^{-1} \hat{\mathbf{J}}_s^\top \hat{\mathbf{E}}_s,
\label{eq:lm_step}
\end{equation}
где $\lambda_s > 0$ -- адаптивный параметр регуляризации, предотвращающий вырожденность матрицы $\hat{\mathbf{J}}_s^\top \hat{\mathbf{J}}_s$ и обеспечивающий глобальную сходимость, а $\mathbf{I}$ -- единичная матрица. Обновление фаз: $\hat{\boldsymbol{\phi}}_{s+1} = \hat{\boldsymbol{\phi}}_s + \mathbf{d}_s$.

Метод демонстрирует превосходство над аналитическими фазами Шрёдера и эвристическими подходами (напр., алгоритм ABC) по следующим показателям:
\begin{itemize}
    \item \textbf{Эффективность}: Значительное снижение минимального, среднего и максимального CF при увеличении числа гармоник $M$.
    \item \textbf{Вычислительная скорость}: Численная сходимость L-алгоритма в десятки раз быстрее эволюционных методов при сопоставимом или лучшем качестве решения.
    \item \textbf{Идентификация FRF}: Сигналы с оптимизированным CF повышают отношение сигнал/шум (SNR) при измерении частотных характеристик.
\end{itemize}

\subsection{Алгоритм с построением виртуальной модели объекта}

Алгоритм измерения предполагает создание на основе данных динамических моделей объекта (цифровых двойников, Digital Twin, DT) \cite{wunderlich2021digital}. Модель должна удовлетворять строгим требованиям: работа в реальном времени без переполнений процессора, учет крупных сигналов (large-signal), усреднение по периоду коммутации (switch-averaged) и возможность встраивания (embeddable) в стандартный цифровой контроллер без использования специализированных решателей. Предложено использовать архитектуру динамической нейронной сети NARX (Nonlinear AutoRegressive eXogenous, рис.~\ref{fig:NARX_ANN}).

\begin{figure}[H]
	\centering
	\includegraphics [width=1.00\linewidth] {review/NARX_ANN.png}

	\caption{Архитектура цифрового двойника из \cite{wunderlich2021digital}}
	\label{fig:NARX_ANN}
\end{figure}

Статические полносвязные сети не подходят для моделирования систем дифференциальных уравнений, так как не имеют памяти. Архитектура NARX решает эту проблему за счет линий задержки на входах и выходах.

Поведение сети в замкнутом контуре (closed-loop) описывается уравнением:
\begin{equation}
    \tilde{y}_k = F(\tilde{y}_{k-1}, \tilde{y}_{k-2}, \dots, \tilde{y}_{k-n}, u_k, u_{k-1}, \dots, u_{k-m}),
\end{equation}
где $u_k$ -- вектор входов системы, $\tilde{y}_k$ -- вектор прогнозируемых выходов модели, $n$ и $m$ -- количество задержек для выходов и входов соответственно, $F(\mathord{\cdot})$ -- нелинейная функция, реализуемая статической ANN.

Для обучения используется разомкнутый контур (open-loop), где на входы задержки подаются истинные измеренные значения:
\begin{equation}
    \tilde{y}_k = F(y_{k-1}, y_{k-2}, \dots, y_{k-n}, u_k, u_{k-1}, \dots, u_{k-m}).
\end{equation}
Такой режим значительно повышает вычислительную эффективность и стабильность процесса оптимизации.

Внутренние нейроны используют нелинейную функцию активации. В работе применяется гиперболический тангенс:
\begin{equation}
    f(x) = \tanh(x) = \frac{2}{1+e^{-2x}} - 1.
\end{equation}

Для обеспечения физической интерпретируемости и универсальности модели входы выбираются как управляющие и внешние возмущения, а выходы -- как состояния системы. На примере повышающего (boost) преобразователя:
\begin{itemize}[leftmargin=*]
    \item \textbf{Вектор входов} $u_k$:
    \begin{equation}
        u_k = \begin{bmatrix} d[k] \\ v_g[k] \\ i_{\text{load}}[k] \end{bmatrix},
    \end{equation}
    где $d[k]$ -- коэффициент заполнения, $v_g[k]$ -- входное напряжение, $i_{\text{load}}[k]$ -- ток нагрузки.
    \item \textbf{Вектор выходов} $y_k$:
    \begin{equation}
        y_k = \begin{bmatrix} i_L[k] \\ v_o[k] \end{bmatrix},
    \end{equation}
    где $i_L[k]$ --- ток индуктивности, $v_o[k]$ --- напряжение на выходном конденсаторе.
\end{itemize}

Обучающая выборка генерируется исключительно в среде компьютерного моделирования, что гарантирует отсутствие шумов измерений и исключает риски повреждения оборудования. Для корректного обучения крупной нелинейной модели данные должны покрывать:
\begin{itemize}
    \item широкий частотный спектр (chirp-сигналы, белый шум) для идентификации динамических характеристик;
    \item медленные синусоиды и ступенчатые функции для охвата различных постоянных рабочих точек (DC operating points);
    \item большие амплитуды возмущений для выявления нелинейных эффектов (например, правого нуля в boost-преобразователе).
\end{itemize}

Сбор данных осуществляется на частоте коммутации (в примере ${f_{sw} = 20}$~кГц).

\needspace{1\baselineskip}
Структура сети и параметры обучения задаются до запуска алгоритма. В работе установлено:
\begin{itemize}[leftmargin=*]
    \item Глубина: 3 скрытых слоя по 10 нейронов в каждом (глубокая архитектура показала лучшую сходимость, чем мелкая);
    \item Задержки: ${n = m = 1}$ (соответствует дискретизации интегральных уравнений состояния первого порядка; увеличение задержек ухудшало аппроксимацию);
    \item Метрика качества -- среднеквадратичная ошибка (MSE):
    \begin{equation}
        \text{MSE} = \frac{1}{N} \sum_{k=1}^{N} \varepsilon_k^2 = \frac{1}{N} \sum_{k=1}^{N} (y_k - \hat{y}_k)^2.
    \end{equation}
    \item Алгоритм оптимизации: Levenberg-Marquardt (гибрид градиентного спуска и метода Ньютона);
    \item Обновление весов (в базовой формулировке градиентного спуска):
    \begin{equation}
        w_{ij}^{\text{new}} = w_{ij}^{\text{old}} - \eta \left. \frac{\partial}{\partial w_{ij}} (\text{MSE}) \right|_{w_{ij}=w_{ij}^{\text{old}}},
    \end{equation}
    где $\eta$ -- скорость обучения. Максимальное число эпох -- 1000.
\end{itemize}

\needspace{1\baselineskip}
Для гарантии надежности цифрового двойника применяется двухэтапная проверка:
\begin{enumerate}[leftmargin=*]
    \item \textbf{Валидация во временной области:} тестирование на независимом наборе данных с экстремальными переходными процессами и широкополосным шумом. Модель точно отслеживает как стационарные режимы, так и быстрые динамические изменения.
    \item \textbf{Валидация в частотной области:} обученная NARX-сеть линеаризуется вокруг заданной рабочей точки, и снимается малосигнальная частотная характеристика (АЧХ/ФЧХ). Результаты сравниваются с аналитическими передаточными функциями преобразователя. Например, для boost-топологии передаточная функция \enquote{управление-выходное напряжение} имеет вид:
    \begin{equation}
        \frac{\hat{v}_o(s)}{\hat{d}(s)} = \frac{D'}{s^2 LC + s R_s C + D'^2},
    \end{equation}
    где $D' = 1-D$, $R_s$ -- эквивалентное последовательное сопротивление (ESR) индуктивности. Совпадение частотных характеристик подтверждает, что сеть выучила физические законы системы, а не переобучилась на временные ряды.
\end{enumerate}

Представленный метод позволяет создавать высокоточные, быстродействующие и легко встраиваемые цифровые двойники силовых преобразователей. Использование динамической NARX-архитектуры с обучением в разомкнутом контуре на синтетических данных обеспечивает корректную аппроксимацию нелинейной динамики без итерационных решателей. Двойная валидация гарантирует надежность модели для задач мониторинга, диагностики, прогнозирования остаточного ресурса (PHM) и программно-аппаратного моделирования (HIL).
